## 🔤 基础语法秀
### 文本样式（一秒抓眼球）
| 语法 | 效果 |
|------|------|
| `**加粗**` | **这是能镇住场的加粗** |
| `*斜体*` | *这是带感的斜体* |
| `~~删除线~~` | ~~这是划掉的旧内容~~ |
| `<u>下划线</u>` | <u>这是突出的下划线</u> |
| `<span style="color:#FF4444">红色字</span>` | <span style="color:#FF4444">这是吸睛的红色文本</span> |
